# Private Party — Wedding Landing Page

Standalone landing page for the Private Party wedding product.

## What is this?
A waitlist/marketing page for the wedding-specific premium tier of Private Party. Validates demand before building the full feature set.

## How to preview
Just open `index.html` in any browser. No server or build step needed.

## How to deploy
Drop `index.html` on any static host (Vercel, Netlify, Cloudflare Pages) or add as a route on privateparty.app.

## Pricing plan
$99 one-time fee for couples. Covers the full wedding planning period.

## Relationship to Private Party
This is marketing for a premium tier — the actual features (private albums, guest list encryption, wedding website) will be built into the main Private Party app when the time comes. The photo privacy library already lives in the PP codebase at `lib/photo-privacy/`.
